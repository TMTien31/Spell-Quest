# Spellbound Gameplay Guide

This document explains the main screens and gameplay flow in Spellbound.

## Main Screens

### Header HUD

The header is visible on core game screens and shows:
- HP (red)
- Coins
- Navigation tabs (Map, Shop, Library in sandbox mode)

### Adventure Map

The map screen shows:
- Current level name
- Encounter path progression
- Current gate/boss node state
- Reset actions (map/soft/hard)

### Combat View

In combat, players:
- Listen to target word pronunciation
- Type letters into segmented inputs
- Use support items (hint, shield, reveal letters, armor plate)
- Defeat gate/boss encounters by spelling correctly

### Library

In sandbox mode, players can:
- Add custom words
- Import comma-separated word lists
- Delete words
- Reset vocabulary list

### Shop

Players can purchase:
- Hint token
- Shield item
- Reveal letter
- Lucky Spin entry
- HP or shield-related items

## Progression

- Complete encounters in order to unlock boss fights.
- Defeat boss to complete the level and move to the next map.
- Levels increase in difficulty by encounter mix and vocabulary.

## Lucky Spin Flow

- Enter Lucky Spin from shop.
- Coins are deducted only when pressing SPIN.
- Player must claim reward before next spin.
- After claiming, player can spin again or exit.

## Reset Modes

- Reset Map: Regenerates current map progression.
- Soft Reset: Restarts journey while keeping existing resources.
- Hard Reset: Resets the run state to initial values.

## Gameplay Tips

- Use pronunciation audio before submitting.
- Save shield items for high-pressure encounters.
- Keep your vocabulary list balanced across easy/medium/hard words.
